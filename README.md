# Weather App 🌦️

A modern weather application built with HTML, CSS and JavaScript.

## Features
- Search weather by city
- Real-time weather data
- Temperature, humidity, wind and visibility
- Modern responsive UI
- Loading and error states

  ## Technologies
- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await
- OpenWeather API
